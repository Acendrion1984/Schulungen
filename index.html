Daten werden als JSON-Datei direkt in einem GitHub Repository gespeichert.

📋 Einrichtung (einmalig, ~3 Minuten):
  1. Kostenloses GitHub-Konto erstellen → github.com
  2. Neues Repository anlegen (privat möglich)
  3. Settings → Developer Settings → Personal Access Tokens → Fine-grained
  4. Token mit Contents: Read & Write Berechtigung erstellen
  5. Token + Repo unten eintragen → Verbinden ✅
